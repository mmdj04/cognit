Preview Release

Version 0.0.1

10 June 2026

Initial preview release. Core compiler (lexer, parser, generator), CLI, and basic examples.

Core Language

  • Let / Var bindings — immutable and mutable variable declarations
  • Functions — first-class functions with arrow syntax
  • Lists and Dicts — array and object literals
  • For / While loops — imperative control flow
  • Import system — module loading

Compiler

  • Lexer — tokenization with indent tracking and string interpolation
  • Parser — recursive descent parser with precedence climbing
  • Generator — JavaScript code generation with runtime injection
