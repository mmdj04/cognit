Latest Release

Version 0.1.0

17 June 2026

Cognit 0.1.0 is the first public release — a general-purpose programming language designed for the future of AI and robotics, compiling to JavaScript.

▶ Try it in the Playground

Language Features

Functional Pipes new

Chain operations with the |> pipe operator. The piped value is passed as the first argument.

let result = numbers
    |> filter(fn(x) -> x % 2 == 0)
    |> map(fn(x) -> x * x)
    |> reduce(fn(acc, x) -> acc + x, 0)

Pattern Matching new

Express complex branching with match and pattern cases.

fn describe(n) {
    match n {
        0 -> "zero"
        1 -> "one"
        _ -> "many"
    }
}

String Interpolation new

Embed expressions directly in strings with {expr} syntax.

let name = "Cognit"
print("Hello, {name}!")  // Hello, Cognit!

Expression-oriented If new

Every if expression returns a value — no ternary operator needed.

let grade = if score >= 90 -> "A"
            else if score >= 80 -> "B"
            else -> "F"

Exponentiation & Matrix Multiply new

Two new operators: ** for exponentiation and @ for matrix multiplication.

let x = 2 ** 10   // 1024
let y = a @ b     // matrix multiply

Tensor & AI

Tensor Operations new

First-class tensor type with arithmetic, matrix multiplication, and shape manipulation.

let a = vec(1, 2, 3)
let b = vec(4, 5, 6)
print(a.add(b).data)    // [5, 7, 9]
print(a.matmul(b).data) // 32

let m = mat([[1, 2], [3, 4]])
print(eye(3).data)      // identity matrix
print(zeros(2, 3))      // zero matrix

Agent Syntax new

Declarative agent configuration blocks for LLM integration.

let assistant = agent {
    model = "gpt-4"
    temperature = 0.7
    instructions = "You are helpful"
}

Runtime

Collection Utilities new

Built-in map, filter, reduce, range, sort, reverse, chunk, unique, zip, flatten, group_by, and more.

JSON Support new

Parse and stringify JSON with json_parse and json_stringify.

Math Functions new

cos, sin, sqrt, exp built into the runtime.

Tooling

REPL new

Run cognit without arguments to start an interactive REPL with .exit to quit.

Error Messages improved

Compile and parse errors now show the exact source line with a ^ caret pointing to the problem.

Playground new

Try Cognit in your browser at mmdj04.github.io/cognit with syntax highlighting, live compilation, and example selector.

Preview Release

Version 0.0.1

10 June 2026

Initial preview release. Core compiler (lexer, parser, generator), CLI, and basic examples.

Core Language

  • Let / Var bindings — immutable and mutable variable declarations
  • Functions — first-class functions with arrow syntax
  • Lists and Dicts — array and object literals
  • For / While loops — imperative control flow
  • Import system — module loading

Compiler

  • Lexer — tokenization with indent tracking and string interpolation
  • Parser — recursive descent parser with precedence climbing
  • Generator — JavaScript code generation with runtime injection
